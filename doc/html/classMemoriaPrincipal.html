◆ load_hilo()

int MemoriaPrincipal::load_hilo ( int  n)

Lee el archivo de instrucciones del hilillo especificado.

Se asume que la ruta del archivo a leer es de tipo .txt y se encuentra en un subfolder de nombre nombreDelFolderContenedor. El metodo abre el archivo y carga sus datos en el vector de instrucciones memoriaInstrucciones. Para saber donde debe escribir la proxima vez que se llame este metodo, lleva el indice en la variable memoriaInstruccionesWritingIndex. Este metodo usa funciones de c como strtok().

Parámetros
nEl numero que se refiere al archivo de instrucciones del hilillo. Empieza en 0 con 0.txt.
Devuelve
La direccion de memoria donde empiezan las instrucciones del hilillo.
